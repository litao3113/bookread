# 书源配置导入说明
需要先安装爱阅书香6.8.1版本
然后在**爱阅书香**书架上的**书源配置**界面上打开**同步功能**。在里面加入本项目地址
```
https://github.com/litao3113/bookread/tree/main/main-01
https://github.com/litao3113/bookread/tree/main/main-02
https://github.com/litao3113/bookread/tree/main/main-03
https://github.com/litao3113/bookread/tree/main/main-04
https://github.com/litao3113/bookread/tree/main/main-05
https://github.com/litao3113/bookread/tree/main/main-06
https://github.com/litao3113/bookread/tree/main/main-07
https://github.com/litao3113/bookread/tree/main/main-08
https://github.com/litao3113/bookread/tree/main/main-09
https://github.com/litao3113/bookread/tree/main/main-10
https://github.com/litao3113/bookread/tree/main/main-11
https://github.com/litao3113/bookread/tree/main/main-12
https://github.com/litao3113/bookread/tree/main/main-X
```
用**获取书源配置**就可以获取所有的书源了。
